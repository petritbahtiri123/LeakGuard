LeakGuard

File Scanner

Local-only scan. Files are not uploaded, sent to APIs, or stored by LeakGuard.

Choose a text file, text PDF, DOCX, XLSX, or image

Text files, text PDFs, DOCX text, XLSX spreadsheet text, and PNG/JPG/JPEG/WEBP images can be scanned locally and exported as redacted text copies. Text PDF scanner results can also export a .redacted.pdf regenerated from sanitized extracted text; it is not layout-preserving, and .redacted.txt remains available as the fallback. DOCX scanner results can also export a .redacted.docx regenerated from sanitized extracted text; it is not layout-preserving, original styles, images, comments, and metadata are not preserved, it does not copy original DOCX XML parts, it does not redact embedded images, .doc, .docm, and macros are unsupported, and .redacted.txt remains available as the fallback. XLSX scanner results can also export a .redacted.xlsx regenerated from sanitized extracted text; it is not layout-preserving, formulas, charts, styles, comments, hidden sheets, metadata, custom XML, calc chains, and media are not preserved, original XLSX XML parts are not copied, .xls, .xlsm, and macros are unsupported, and .redacted.txt remains available as the fallback. Protected-site DOCX output can hand off a regenerated .redacted.docx when complete; truncated or unsafe DOCX regeneration falls back to .redacted.txt or blocks raw upload. Protected-site XLSX output can hand off a regenerated .redacted.xlsx when complete; truncated or unsafe XLSX regeneration falls back to .redacted.txt or blocks raw upload. Protected-site text PDF output can hand off a regenerated .redacted.pdf when complete; truncated or unsafe PDF regeneration falls back to .redacted.txt or blocks raw upload. Image OCR is English-only, runs only after you select an image and click Scan File, and is limited to image files on this scanner page. Scanner image visual redaction outputs a flattened PNG with black rectangles when OCR boxes are available; JPG, JPEG, and WEBP inputs are not preserved as their original format in this release. Protected-site upload OCR is on by default for supported image uploads, can be turned off in settings, and exports a flattened redacted PNG only when OCR box confidence is eligible; otherwise LeakGuard falls back safely or blocks raw image upload. XLSX formulas are scanned as text only and are not executed. Scanned PDF OCR, layout-preserving PDF/DOCX/XLSX redaction, image format preservation, macros, legacy DOC, DOCM, legacy XLS, XLSM, embedded media, archive, executable, and binary-file OCR are not enabled in this release.
