LeakGuard

File Scanner

Local-only scan. Files are not uploaded, sent to APIs, or stored by LeakGuard.

Choose a text file, text PDF, DOCX, XLSX, or image

  • Text files, text PDFs, DOCX text, XLSX spreadsheet text, and PNG/JPG/JPEG/WEBP images are scanned locally.
  • Exports redacted text copies. Text PDF, DOCX, and XLSX scans can also export regenerated .redacted.pdf, .redacted.docx, or .redacted.xlsx files.
  • Regenerated PDF/DOCX/XLSX files are not layout-preserving; .redacted.txt remains available as the fallback.
  • DOCX regeneration does not copy original DOCX XML parts and does not preserve styles, images, comments, or metadata.
  • XLSX formulas are scanned as text only and are not executed. Formulas, charts, styles, comments, hidden sheets, metadata, custom XML, calc chains, media, and original XLSX XML parts are not preserved.
  • Protected-site PDF, DOCX, and XLSX output can hand off regenerated redacted files when complete; truncated or unsafe regeneration falls back to .redacted.txt or blocks raw upload.
  • Image OCR is English-only, runs only after you select an image and click Scan File, and is limited to image files on this scanner page.
  • Scanner image visual redaction outputs a flattened PNG when OCR boxes are available. JPG, JPEG, and WEBP inputs are not preserved as their original format.
  • Protected-site upload OCR is on by default for supported image uploads, can be turned off in settings, and exports a flattened redacted PNG only when OCR box confidence is eligible.
  • Not enabled in this release: Scanned PDF OCR, layout-preserving PDF/DOCX/XLSX redaction, image format preservation, macros, legacy DOC/DOCM, legacy XLS/XLSM, embedded media, archive, executable, and binary-file OCR.
